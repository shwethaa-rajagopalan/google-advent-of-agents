# Design Documents

Technical design specifications and architectural decisions for Scion.

## Structure

- **`hosted/`** - Distributed architecture specs (Hub, Runtime Brokers, frontend, auth flows).
- **`kubernetes/`** - Kubernetes deployment and agent sandboxing designs.
- **`metrics/`** - Observability and OpenTelemetry integration designs.
- **`_archive/`** - Historical/superseded design documents.
- **Top-level `.md` files** - Core design docs for local architecture and CLI features.

## Usage

These documents serve as the source of truth for architectural decisions. Review the `hosted/` directory for the latest distributed platform architecture.
