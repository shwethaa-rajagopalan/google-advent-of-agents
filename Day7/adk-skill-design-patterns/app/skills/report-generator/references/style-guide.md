# Report Style Guide

## Tone
- Write in third person ("The system demonstrates..." not "I found...")
- Be concise — one idea per paragraph
- Use active voice where possible
- Avoid jargon unless the audience is technical

## Formatting
- Use H2 (`##`) for major sections, H3 (`###`) for subsections
- Use bullet lists for 3+ items, numbered lists for sequential steps
- Include a summary table for quantitative findings
- Bold key terms on first use

## Structure Rules
- Executive Summary must be under 150 words
- Each section must have at least one concrete finding or recommendation
- End with "Next Steps" — actionable items, not vague suggestions
- Include a "Methodology" section if data analysis was performed
