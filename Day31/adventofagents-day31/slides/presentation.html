Why A2UI?

The declarative standard for Agent-to-User Interfaces.

What is A2UI?

A2UI is a declarative UI protocol that allows agents to send structured JSON instead of raw code.

  • Data, Not Code: Eliminates malicious script execution by design.
  • Native Performance: Renders as native widgets (React, Flutter, SwiftUI).
  • LLM-Friendly: Optimized for real-time streaming and incremental generation.

What A2UI is NOT

  • Not a Framework: It's a cross-platform protocol for UI declaration.
  • Not an HTML Replacement: Built for agent UIs, not for static websites.
  • Not a Styling System: The client controls the brand; the agent provides the structure.
  • Not Web-Only: Designed to work natively on iOS, Android, & Desktop.

Enterprise Secure

Logic and credentials never leave your Cloud boundary. Gemini only sees the declarative UI it needs to render natively.

Verified Discovery

Enterprise-grade auth via A2A ensures only trusted agents can communicate with your workspace.
